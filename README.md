# Flight Delay Compensation Claim

This project helps users submit and track compensation claims for delayed or cancelled flights under EU261 and DOT regulations.

## Technologies Used
- HTML
- CSS
- JavaScript
- React.js
- Node.js
- Express.js
- MySQL

## Features
- Claim Form
- Document Upload
- Eligibility Checker
- Claim Status Tracking
- Admin Review
- Email Notifications
